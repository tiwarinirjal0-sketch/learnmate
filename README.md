# LearnMate — AI-Powered Study Assistant

LearnMate is an intelligent AI-driven learning platform designed to enhance the way students study, understand, and retain information. The application enables users to interact with their learning materials through natural language conversations, transforming static documents into an interactive and personalized learning experience.

Built with modern web technologies and powered by the Gemini API, LearnMate provides AI-assisted tools such as document-based conversations, automated summaries, note generation, quizzes, and flashcards to help students improve productivity, comprehension, and long-term memory retention.

## Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **AI Model Integration:** Google Gemini API

## Core Features

* **AI Document Interaction:**
  Allows students to chat with their study materials and receive context-aware answers using large language models.

* **Intelligent Content Summarization:**
  Converts lengthy educational content into concise, easy-to-understand summaries for faster revision.

* **Automated Note Generation:**
  Generates structured and organized notes from learning resources, reducing manual effort and improving study efficiency.

* **AI-Generated Quizzes:**
  Creates personalized quizzes from study content to evaluate understanding and reinforce learning.

* **Smart Flashcard Generation:**
  Produces flashcards for active recall and improved information retention.

* **General AI Learning Assistant:**
  Provides an interactive chatbot interface for answering questions, explaining concepts, and assisting students throughout their learning journey.

## Project Highlights

* Integrated Gemini API to deliver intelligent and context-aware AI responses.
* Developed a full-stack application with a scalable React.js frontend and Node.js/Express backend architecture.
* Implemented AI-powered learning workflows to transform traditional study methods into an interactive experience.
* Designed the platform with a focus on improving student productivity, engagement, and knowledge retention.
